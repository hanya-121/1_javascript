반복가능항목

- 반복가능한 객체(배열)임.
- for ...of : 문자열 요소를 반복 할 수 있음
- 일련의 요소를 반복하는 것을 의미
- 배열 → 내장배열 : 정해져 있는 반복
        외장배열 : 정해지지 않은 배열을 사용할 때
- 문자열 반복 / 배열 반복

Set 집합

- 고유한 값의 컬렉션
- 각 값은 set에서 한 번만 발생 할 수 있음
- 중복 허용 불가

Map

- 키와 값을 쌍으로 가지고 있음
- 키는 모든 데이터의 유형
- 맵은 원래의 삽입 순서를 기억
- 여러 개의 값을 가지고 있는데 Map의 값을 가져올 때는 get()이라는 메서드를 사용하여 가져옴
