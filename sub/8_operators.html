Operators 연산자

- 할당 연산자 = 값을 할당 
            ex) let x = 10
- 덧셈 연산자 + 값을 더함 
            ex) let x = 5; let y = 2; let z = x + y 
- 곱셈 연산자 * 값을 곱함
            ex) let q = 5; let w = 5; let multi = w * q
- 비교 연산자 > or < 변수를 선언 후 값을 비교
