HTML 이벤트

HTML이벤트는 브라우저가 수행하는 일이거나 사용자가 수행하는 일 일 수 있음
- 페이지 로딩이 완료 되었습니다
- 입력 필드가 변경 되었습니다
- 버튼이 클릭 되었습니다



특정 이벤트가 발생 했을 때 코드가 실행 됨
- onchange  : HTML요소가 변경 될 때
- onclick   : 사용자가 html요소(엘리먼트)를 클릭 할 때
- onmouseover:사용자가 html요소 위로 마우스를 이동 할 때 (hover)
- onmouseout: 사용자가 html요소 밖으로 나갔을 때
- onkeydown : 사용자가 키보드 키를 눌렀을 때
- onload    : 브라우저가 페이지 로딩을 완료된 이후
